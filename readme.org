* SRFI 33: Integer Bitwise-operation Library

** by Olin Shivers

This repository hosts [[http://srfi.schemers.org/srfi-33/][SRFI 33]]: Integer Bitwise-operation Library, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /withdrawn/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-33/srfi-33.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-33/][join the SRFI-33 mailing list]] and send your message there.
